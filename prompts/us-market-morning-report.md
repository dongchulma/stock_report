# 미국 증시 프리마켓 리포트 (매일 08:20 KST)

## 목적
한국 장 개장 전, 전일 밤 미국 증시 마감 현황과 핵심 지표를 정리해 텔레그램으로 전송한다.

## 공통 규칙 (반드시 지킬 것)
1. 리포트는 반드시 한국어로 작성한다.
2. 리포트에 들어가는 모든 숫자 지표(지수, 종목 가격, 환율, 원자재, 금리 등)는 전일 대비 변동률(%)을 함께 표기한다. 변동률을 구할 수 없으면 그 사실을 명시한다.
3. 각 지표는 숫자만 나열하지 않고, 대화체로 짧은 설명을 덧붙인다 (예: "코스피가 어제보다 1.5% 올랐네요, 반도체주가 끌어준 덕분이에요.").

## 포함할 내용
1. 전일 밤 미국 증시 마감 현황: 다우존스/나스닥/S&P500 종가·등락률, 필라델피아반도체지수(SOX), VIX, 주요 특징주/섹터 동향, 핵심 매크로 이슈(연준 발언, 경제지표, 지정학 이슈 등)
2. 한국 주요 지수 (전일 종가 기준 — 오늘 장 시작 전): KOSPI, KOSDAQ
3. 미국 국채금리: 2년물, 10년물, 30년물 + 10Y-2Y 스프레드
4. 반도체 섹터: DRAM/NAND/HBM 현물가 동향, 한국 반도체주(삼성전자·SK하이닉스·한미반도체·HPSP·리노공업 등) 전일 종가/등락률
5. 하이퍼스케일러 Capex 동향: Amazon/Google/Meta/Microsoft 최신 발표
6. 환율 & 원자재: USD/KRW, WTI, Gold, DXY
7. 수급 분석 (전일 마감 기준): 외국인/기관/개인 순매수·순매도
8. 오늘(한국시간 기준) 주목할 주요 일정/이벤트 (있는 경우)

데이터는 WebSearch/WebFetch로 최신 정보를 조사해서 채운다. 실시간 수치를 구할 수 없으면 가장 최근 확인 가능한 값을 쓰고 기준 시점을 표기한다.

## 포맷
- 한국어, 텔레그램 메시지로 읽기 좋게 작성 (굵게는 `*텍스트*`, 폭이 넓은 표 대신 글머리 기호 위주)
- 섹션 제목 + 구분선(━━━)으로 가독성 있게 구성

## 실행 절차
0. **휴장일 확인 (가장 먼저, 반드시 자동으로 판단할 것)**: 오늘이 한국 증시 휴장일(주말 또는 공휴일)인지 WebSearch로 확인한다.
   이 작업은 headless로 실행되어 사용자에게 질문할 수 없다 — 휴장 여부를 스스로 판단하고, 절대 사용자에게 되묻지 않는다.
   한국 증시가 휴장이면 리포트 작성과 텔레그램 전송을 모두 건너뛰고, 곧바로 아래를 실행한 뒤 종료한다:
   ```
   python3 scripts/update_status.py us-market-morning <YYYY-MM-DD> skipped_holiday
   ```
   개장일이면 아래 1~4단계를 정상적으로 진행한다 (전일 밤 미국 증시가 휴장이었던 경우엔, 1번 항목에 "미국 증시는
   휴장이었다"고 명시하고 나머지 지표는 정상적으로 채운다 — 이 경우는 건너뛰지 않는다).
1. 리포트를 로컬 파일(예: `/tmp/report.md`)로 작성한다.
2. GitHub push는 하지 않는다. 네이버 블로그 포스팅도 하지 않는다.
3. 텔레그램으로만 전송한다 (환경변수 `TG_BOT_TOKEN`, `TG_CHAT_ID`는 이미 설정되어 있음):
   ```
   python3 scripts/send_telegram.py /tmp/report.md
   ```
4. 상태를 기록한다 (환경변수 `GH_TOKEN`, `GH_REPO`는 이미 설정되어 있음):
   ```
   python3 scripts/update_status.py us-market-morning <YYYY-MM-DD> success
   ```
   실패한 단계가 있으면 `success` 대신 `failed`로 기록한다.
